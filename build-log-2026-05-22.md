### Nav Always-Navigate Fix (Real Fix)

**Problem:** Previous approach used `router.push('/module?t=' + Date.now())` — Next.js does not re-mount components for same-path navigations with different query params, so this never worked.

**Fix implemented:**
- `AppShell.jsx` — `go()` function updated:
  ```js
  const go = path => {
    if (router.asPath === path || router.asPath.startsWith(path + '/') || router.asPath.startsWith(path + '?')) {
      router.replace(path).then(() => router.reload());
    } else {
      router.push(path);
    }
  };
  ```
- `SedonaCRM.jsx` — `handleNav()` function added (same logic); replaces all 7 `router.push('/xxx?t=Date.now()')` calls for: Tenants, Suites, Issues, Contacts, Vendors, Owners, Rent Schedule.
- `CLAUDE.md` Development Rule #3 updated to document the real pattern and explicitly warn against `?t=Date.now()`.
